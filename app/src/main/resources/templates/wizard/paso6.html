Paso 6 – Despliegue en EC2

Ver guía de este paso

En este paso se guardan las variables necesarias para desplegar en una instancia EC2. La llave SSH es un secreto y no se precarga por seguridad.

IP pública o dominio. Sin http:// ni https://.
Usuario SSH típico: ubuntu (Ubuntu), ec2-user (Amazon Linux).
Puerto público donde se accederá a la app.
Línea(s) de known_hosts para evitar prompts de SSH en CI/CD.
No se precarga por seguridad. Si el paso ya estaba OK y recargas, tendrás que pegarla de nuevo si quieres verla en el panel.
El fichero descarga comandos para preparar una EC2 con Docker, Docker Compose y AWS CLI.

Consejo para crear los Secrets:
  • GitHub → Settings → Secrets and variables → Actions → New repository secret
  • GitLab → Settings → CI/CD → Variables → Add variable
  • Jenkins → Manage Jenkins → Credentials → (Global/Folder) → Add Credentials
